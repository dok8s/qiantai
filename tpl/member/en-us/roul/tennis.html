TENNIS

Last Update: 25/9/2012

1. GENERAL RULES

  • If the scheduled venue is changed to a different surface, all bets will be considered void.
  • All bets will be considered valid only when the match is finished, unless explicitly stated below or in the individual Bet Type rules.
  • If a player fails to compete in any tournament or match, all bets on that player (including tournament or individual match markets) will be voided.
  • If a player or pairing retires or is disqualified during a match before its completion, all bets for the match will be voided, unless explicitly stated below or in the individual Bet Type rules.
  • If the scheduled duration of a match is reduced or if there is an increase in the number of points to win, all bets will be voided.
  • If a match is postponed or suspended all bets are still considered valid if the match is completed.
  • Where markets are offered on individual games or sets within a match, a retirement or disqualification during a game or set will render bets on that game or set market and all individual game or set markets void, unless explicitly stated in the individual Bet Type rules.
  • If a match starts before schedule, only the transactions before the match starts will be considered valid. The transactions after the match starts will be considered invalid. (Except in-play bet types).
  • For the four championships (Australian Open, US Open, French Open, Wimbledon), the way to determine the result of match is:
    • For men's match: the player, who wins 3 sets first, wins the match.
    • For women's match: the player, who wins 2 sets first, wins the match.

2. BET TYPES

Below is a breakdown of the rules for the individual bet types we offer for Tennis.

2.1. Money Line

  • Predict who will win the match. This market will contain the two players.

2.2. Set Handicap

  • Predict who will win the match with the indicated set handicap applied.

2.3. Game Handicap

  • Predict who will win more games in a specified period (e.g. Match, 1st Set, 2nd Set) after applying the indicated handicap.
  • A tie-break is considered as one game.

2.4. Game Over / Under

  • Predict whether the total number of games played in a specified period (e.g. Match, 1st Set, 2nd Set) will be over or under the indicated total line.
  • A tie-break is considered as one game.
  • If a match is abandoned, Over / Under bets will only be settled when the market has been unconditionally determined as any further potential games have no affect on the market result. In all other scenarios, bets will be considered void.

2.5. Odd / Even

  • Predict whether the total number of games played in a specified period (e.g. Match, 1st Set, 2nd Set) will be odd or even.
  • A tie-break is considered as one game

2.6. Set Betting Markets

  • Predict the outcome of the specified set.
  • The bets will still stand once the specified set has been completed.
  • The bets on the set betting markets will be considered void if a player retires before the specified set is completed.

2.7. Set Betting – Correct Score

  • Predict the final sets score of the match.
